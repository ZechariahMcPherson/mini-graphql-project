# mini-graphql-project

Building out mini graphql project via [this tutorial](https://www.youtube.com/playlist?list=PL4cUxeGkcC9iK6Qhn-QLcXCXPQUov1U7f)

## How to run project

### Install node modules

```
yarn install
```

### start project

```
yarn start
```

### url

```
http://localhost:4000/graphql?
```
